# LiteBIRD scanning hitmap 
This code simulate the LiteBIRD scanning strategy (starting from 1/1/2033) and generate the hit-map for the selected source among mercury, venus, mars, jupiter, saturn and uranus. The time sampling is 1 second. The default parameters for the scanning are those selected from the LiteBIRD collaboration at the present date.
It is possible to select the source and to implement the Lissajous orbit. The default parameters for the Lissajous orbit are the same selected for the WMAP mission.
